## 项目描述

**1、项目简介**

假设有一个软件每隔一小段时间会记录一次用户的搜索记录，记录为英文。

输入文件和输出文件以命令行参数传入。对输入文件进行词频统计。

**2、如何运行**

进入221801328\src目录，输入以下代码

编译

javac -encoding UTF-8 WordCount.java Lib.java

执行，其中input.txt和output.txt可以自行指定

java WordCount input.txt output.txt
