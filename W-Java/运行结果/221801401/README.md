项目主要实现了文件的字符数、行数、单词数以及出现次数top10的统计，从命令行读取两个文件名作为输入文件和输出文件，
先将文件中的数据按字符串读出，再用四个函数方法对这个字符串进行统计，最后将统计好的数据输出到出输出文件中。
