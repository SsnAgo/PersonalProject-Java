## 如何运行
>先从cmd窗口进入src文件夹，利用javac WordCount.java 再使用java WordCount input.txt output.txt来运行项目。

## 功能简介
>1. 对输入的input.txt的文本进行字符数统计。
>2. 对文本的单词数统计
>3. 对文本的有效行数进行统计
>4. 对统计频率最高的10个单词进行输出，如果同频率则按字典序小的在前。
